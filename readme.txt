== Description ==

This `Auto Product After Upload Image` plugin will let you create single/bulk product after uploading any media from wordpress media gallery. This is very much essential plugin for photo blog or where there are a lots number of image producting in a wordpress driven site.

When you will upload an image from wordpress media gallery then a product will be created automatically with that image as featured image.

Feature functions: 
1. Auto Thumbnail, gallery.
2. Auto pick Category, tag, custom taxonomies, rate, featured, product visible.
3. Good working with variation product.

== Installation ==

To **Install** this `Auto Product After Upload Image` plugin there is no any complexity. It's very simple like other plugin. Just follow the procedure described below.

1. Active Plugin
2. Active Woocommerce
3. Global Options: Go Dashboard -> APAUI Creator - Auto Apply for new Product Created
4. Single Options: Please have a look Plugin Screenshot before use plugin.

== Frequently Asked Questions ==

= This is useful for creating bulk Woocommerce product? =

Yes, of course. The main intention to create this plugin to use where user need bulk image producting and creating product.

= How's it work? =

The plugin checks theme Options and the image name is uploaded, then creates the product with the image and data such as the image name.ZZ
